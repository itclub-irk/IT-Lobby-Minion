# IT-Lobby-Minion


## Запуск на локальной машине

* устанавливаем poetry
* ставим зависимости
```bash 
poetry install
```
* создаём .env и копируем параметры из .env.example
    * токен для бота можно получить в https://t.me/BotFather
    * свой id можно узнать у https://t.me/useridinfobot
### запуск бота:
```bash 
python run.py
```
